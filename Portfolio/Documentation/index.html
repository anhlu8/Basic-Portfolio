Documentation

Please read this first

Resume - single page portfolio/vcard theme is a clean and modern portfolio/vcard, responsive, single page theme built with HTML5 and CSS3.

This documentation is meant for people with little or no knowledge in HTML 5, CSS 3 and JavaScript. Advanced users might find themselfs comfortable directly editing the template since it has a clean, straightforward and easy to modify markup.

For support, questions and other type of assistance, please contact me at tonybogdanov@gmail.com.

Layout

The template uses a responsive layout with maximum width of 960px.

Header

Changing the name and profession

In order to change the most important thing in the whole template - the name and profession in the header you will have to open the included PSD file (with Adobe Photoshop), and edit the info there. You will find the desired entry in header > logo. After the editing is completed simply re-export the logo and replace ./img/logo.png. You do need some advanced knowledge in using Photoshop, and of course a licensed copy of the software.

Changing menu entry labels & urls

Open ./index.html and locate <ul class="navigation">. There inside each li you have a link with attribute data-original-title - change the value for the tooltip text, and inside a span element with class title - change the text to change the menu label. You wont need to change the urls here or you will lose the single page scroll effect. Only the last anchor has an empty href where you should put the path to your downloadable resume.

Home

Changing the featured design entry

The homepage contains a single featured design entry. To change it open ./index.html and locate <h3>Featured in <a , there you can change the path of the image and portfolio section to the desired ones. Change portfolio section link href to # and the id of the section (explained later). Change the data-featured-gallery attribute to # and the id of the portfolio section gallery (explained later). Change the data-featured-index to a number identifying which item should be open from the desired portfolio gallery. F.e. if you want to identify the first element, put 0, if you want the second, put 1 etc.

Section heading

Changing section heading left & right texts

To change the text locate a h1 element and simply edit the text in the left & right parts.

About me

Changing the text

To change the text simply locate the about me section marked with a comment like <!-- ABOUT ME --> and edit the content.

Female head

There is a female version of the code head graphic used in this section, it is located in ./img/code-head-female.png. To use it, simply rename the original to something else and then rename the female version back to ./img/code-head.png.

Skills

Changing the text

Simply locate the skills section marked as <!-- SKILLS --> and edit the text.

Editing the skills

You already have 10 skills divided in two columns, to edit them simply locate each and change the URL and text of the skill. You can also edit the data-original-title attribute for the tooltip. To change the level of the skill locate data-progress and change the value to something else (in percent of course).

Changing work experience

Editing your work experience is as easy as adding / removing / editing the contents of the li tags inside <ul class="work"> section. Just use the provided example. If you place too many entries they will translate on the next row, which would change the original design of the template, but will still look good.

Projects

Changing the projects

You can have as many galleries as you like, just don't forget to use unique ids for the portfolio section & gallery. Each project lives inside a li element and can have a thumbnail in class="thumbnail" and different markup in the class="info".

Contacts

Changing the text

To change the contact text locate the section marked as <!-- CONTACTS --> and edit the text. The form doesn't need any modifications.

Footer

Change the footer copyright & social icons

Locate the footer element and edit the copyright info and/or social icons as you wish. Included are 3 icon styles, but you can always create/buy/find more, just remember that they need to be 20px by 20px.

Background

You can always change the background of the header & footer by specifying a class for the header and footer tags. All available classes can be found in css/style.css or on the preview page.
